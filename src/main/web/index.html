Vygenerujte si vlastní billboard, který můžete sdílet!

Poslední dobou je pro diskutující čím dál náročnější oslovit své protivníky a přesvědčit je o svém názoru. Jednoduché heslovité nástroje jsou sice účinné, ale po čase omrzí a je pracné je vytvořit. Proto stačí využít náš generátor.

Zvolte si, jestli máte rádi prezidenta Zemana nebo ne, a vše ostatní už za vás udělá!

Upozorňujeme, že tento projekt nemá za cíl někoho pošpinit, ale chce recesistickou formou poukázat na některé nevhodné a negativně laděné kampaně. Texty jsou založené na existujících heslech z internetu. Inspirací byl také náhodný slušný čech.

Chcete-li přispět svými hesly, můžete udělat pull request nebo komentář na githubu.
